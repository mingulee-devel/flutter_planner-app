# planner_todo

플러터를 이용한 스터디플래너 어플을 만들었습니다.

열정품은 타이머 어플을 보고 만들어본 것으로,
4학년 1학기 졸업작품으로 약 3~4개월 간 만든 어플리케이션입니다.


- 어플리케이션 설명 ppt 자료도 포함되어 있습니다.
